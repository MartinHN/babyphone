For best background reliability, install this as an app (button above, or "Add to Home Screen"/"Share → Add to Home Screen"). When you tap "Connect" you'll be asked for notification permission — allow it, since that's what puts a persistent "Mic Stream" notification with a Stop button in your notification shade while connected. Android Chrome generally keeps audio playing when you switch apps or lock the screen. iOS Safari is much more restrictive about background web audio — keeping the screen awake (toggle above) is the most reliable way to avoid interruptions there.

Relay server address is the host:port your signaling server prints on startup (e.g. 192.168.1.42:3000) — any device on the LAN can run it. If you're opening this page directly from that server it's filled in automatically; if you're using a version hosted elsewhere (like GitHub Pages), enter it manually — it's remembered for next time, same as the access token. The very first time, open https://<that-address> directly in this browser once and accept the certificate warning, or connections below will fail silently.
